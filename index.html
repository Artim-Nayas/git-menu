Self-updating

Install in a minute

Git Menu uses the GitHub CLI for auth — no tokens to paste.

1 · GitHub CLI

brew install gh
gh auth login

2 · Git Menu

brew install --cask \
  artim-nayas/tap/git-menu

Prefer a direct download? Grab the .dmg from Releases. The app is ad-hoc signed (not notarized); the cask clears the quarantine for you.
